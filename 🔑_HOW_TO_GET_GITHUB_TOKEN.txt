Q: Which scopes should I select?
A: At minimum, check "repo" (and all boxes under it). This gives 
   full control over repositories.

Q: What expiration should I choose?
A: For development: 90 days or No expiration
   For security: 30 days (you'll need to regenerate monthly)

Q: I forgot to copy my token!
A: You'll have to delete it and generate a new one. GitHub won't 
   show it again for security reasons.

Q: Can I use the same token for multiple projects?
A: Yes! One token can work for all your repositories.

Q: Is it safe to save my token?
A: Save it in a secure password manager. Don't commit it to Git 
   or share it publicly.

Q: What if my token expires?
A: Generate a new one and use it. Old token stops working.
